# v1.0.1 — 截圖工具優化

**發布日期：** 2026-02-08

---

## 變更內容

### 修改檔案
- `get_region_coords.py` — 重寫，新增一鍵截圖功能

### 新增功能
- **一鍵截圖**：瀏覽器開啟後，在終端按 `Enter` 即可截圖保存
- **連續截圖**：可反覆按 Enter 截取多張圖片，不需重啟工具
- **重載截圖**：輸入 `r` 可重新載入頁面後再截圖
- **自動命名**：檔名含時間戳，格式 `screenshot_20260208_164500_1.png`
- **截圖資料夾**：所有截圖統一存放在 `screenshots/` 資料夾
- **多 URL 選擇**：當 `game_config.json` 有多個 enabled 遊戲時，提供選單選擇

### 操作方式
```
python get_region_coords.py              # 自動從 game_config.json 讀取
python get_region_coords.py "https://..."  # 指定 URL

操作：
  Enter  → 截圖
  r      → 重新載入頁面後截圖
  q      → 結束
```

### 新增資料夾
- `screenshots/` — 手動截圖存放位置

## 回滾方式
如需回滾，還原 `get_region_coords.py` 到 v1.0.0 版本即可，`screenshots/` 資料夾可保留或刪除。
